Computer Science Principles

Computer Science Principles is a broad computer class dealing with the understanding of basic machine language to complex languages. It touches on App Development, Binary Process, HTML, CSS, PHP, Block Programming, Python, and Informatics. (Course prerequisite Java, C#, C++, Visual Basic, and Web Design).

Cyber Security

Cyber Security is where you get hands on experience with how vulnerable everyday systems are with an understanding of the ethical component behind it, you will see how systems are infiltrated and how to protect yourself and those systems. (Course prerequisite- Computer Science Principles, and one of the following Java, C#, C++, Visual Basic, and Web Design).

Java Programming

Learn the intro into Java programming, using Exclipse IDE as a text editor you learn how the basics of Java.

C# Programming

C# (pronounced as see sharp) The used for GUI frameworks for Windows applications is Winforms and WPF together with .NET Framework. This is a beginner class into the use of C# and its functionality.

Fundamentals of Web Design

This course is designed for beginner Web Developers. You will learn HTML5, CSS, and JavaScript. The students will use ATOM.IO text editor and learn the elements into front end developers.

Visual Basic

Visual Basic is a programming environment from Microsoft in which a programmer uses a graphical user interface (GUI) to choose and modify preselected sections of code written in the BASIC programming language. The IDE we use in class is Visual Studio

C++ Game Programming

C++ is a general-purpose object-oriented programming (OOP) language, developed by Bjarne Stroustrup, and is an extension of the C language. It is therefore possible to code C++ in a "C style" or "object-oriented style. In the course we use the IDE Visual Studio and it is taught using game theory.
